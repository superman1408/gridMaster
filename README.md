# gridMaster
 An interactive Tic-Tac-Toe game built using React.js that demonstrates core library principles like state management, component-based architecture, and lifting state. It features a responsive 3x3 grid where players take turns as 'X' and 'O', with the system automatically calculating the winner or identifying a draw.
Key Features:
Real-time Winner Detection: Automatically scans for horizontal, vertical, and diagonal win patterns after every move.
Game History (Time Travel): Users can view a list of previous moves and "jump back" to a specific point in the game to try different strategies.
Persistent State: Uses React Hooks like useState to maintain the current board status and move sequence.
Reset Functionality: A single-click reset button to clear the board and start a new match instantly.
